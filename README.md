# Field Service Software Pricing Index 2026

This repository contains the raw data for the **Field Service Software Pricing Index**, maintained by the [FieldServiceTools.com Editorial Team](https://fieldservicetools.com).

## About This Dataset
This dataset tracks pricing, verification status, and support channels for major Field Service Management (FSM) software. It is designed to provide transparency and verify the claims made in our research articles.

- **Methodology:** [How We Review](https://fieldservicetools.com/how-we-review/)
- **Live Index:** [View Full Report](https://fieldservicetools.com/research/field-service-software-pricing-index/)
- **Last Updated:** 2026-01-29

## Contents
- `data.csv`: Structured data including pricing models, verification sources, and integration types.
